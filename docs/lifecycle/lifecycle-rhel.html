# Red Hat Enterprise Linux operating system

This solution is built using Red Hat Enterprise Linux (see #rhel-components-table-content) as the base operating system. When upgrading the operating system on the VMs, first verify that the OS version is compatible with Docker EE by looking at the Docker OS compatibility matrix.
